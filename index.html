Vanilla CSS Gallery

A simple image gallery made purely out of CSS. No JavaScript. Suitable for when you need a really simple and lightweight solution.

Necessary CSS

.gallery-nav {
  position: relative;
}

.gallery-item {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  opacity: 0;
  z-index: -1;
}

.gallery-close {
  display: none;
}

.gallery-item:target {
  opacity: 1;
  position: fixed;
  z-index: 2;
}

.gallery-item:target ~ .gallery-nav {
  z-index: 3;
}

.gallery-item:target ~ .gallery-nav .gallery-close {
  display: block;
}
