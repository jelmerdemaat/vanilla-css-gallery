Vanilla CSS Gallery

A simple image gallery made purely out of CSS. No JavaScript. Suitable for when you need a really simple and lightweight solution.

Necessary CSS

.gallery-nav {
  position: relative;
}

.gallery-item {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  opacity: 0;
  z-index: -1;
}

.gallery-close {
  display: none;
}

.gallery-item:target {
  opacity: 1;
  position: fixed;
  z-index: 2;
}

.gallery-item:target ~ .gallery-nav {
  z-index: 3;
}

.gallery-item:target ~ .gallery-nav .gallery-close {
  display: block;
}

Additional CSS

No prefixes included.

.gallery-nav {
  border-radius: 5px;
}

.gallery-item {
  background: black;
  background: rgba(0, 0, 0, 0.85);
  transition: all 0.2s ease-out;
}

.gallery-item img {
  max-width: 90vw;
  max-height: 90vh;
}

.gallery-item:target ~ .gallery-nav {
  top: 25px;
  left: 50%;
  transform: translateX(-50%);
  text-align: center;
  position: fixed;
}

.gallery-item:target ~ .gallery-nav li {
  margin: 0 5px 0 0;
}

}

.gallery-item:target ~ .gallery-nav .gallery-close {
