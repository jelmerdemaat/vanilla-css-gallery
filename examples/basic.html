Vanilla CSS Gallery - Basic

Example of the CSS-only image gallery without any styling. These are the bare basics.

View example with additional styling - View on GitHub - By @jelmerdemaat

Necessary CSS

.gallery-nav {
  position: relative;
}

.gallery-item {
  position: absolute;
  top: 0;
  right: 0;
  bottom: 0;
  left: 0;
  opacity: 0;
  z-index: -1;
}

.gallery-close {
  display: none;
}

.gallery-item:target {
  opacity: 1;
  position: fixed;
  z-index: 2;
}

.gallery-item:target ~ .gallery-nav {
  z-index: 3;
}

.gallery-item:target ~ .gallery-nav .gallery-close {
  display: block;
}

Necessary HTML

<div class="gallery-item" id="1">
	<img src="your/image/source-large.jpg" alt="Tech image">
</div>
<div class="gallery-item" id="2">
	<img src="your/image/source-large.jpg" alt="Nature image">
</div>
<div class="gallery-item" id="3">
